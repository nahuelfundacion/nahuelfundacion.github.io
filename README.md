# Fundación N.A.H.U.E.L.

Sitio web institucional de la **Fundación N.A.H.U.E.L. – Arte, Territorio y Bienestar Regenerativo**.

## Descripción

La fundación promueve iniciativas de arte, cultura, educación, participación ciudadana, restauración patrimonial, ecología y desarrollo territorial con enfoque comunitario.

## Sitio

Este repositorio contiene una primera versión estática de la página institucional, preparada para publicarse mediante GitHub Pages.
